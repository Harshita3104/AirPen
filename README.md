# AirPen
Computer vision application based on hand gesture tracking using Python, OpenCV, and MediaPipe.
AirPen uses computer vision and hand tracking to recognize gestures and enable real-time applications through a webcam.

# Features
1.Real time tracking and gesture recognition.
2.Webcam based interaction. 
3.Gesture controlled interface.
4.Touchless user experience.

# Technologies Used
1.Python
2.OpenCV
3.MediaPipe
