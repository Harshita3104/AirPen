# AirPen
Computer vision application based on hand gesture tracking using Python, OpenCV, and MediaPipe.
AirPen uses computer vision and hand tracking to recognize gestures and enable real-time applications through a webcam.

## Features
- Real time tracking and gesture recognition.
- Webcam based interaction. 
- Gesture controlled interface.
- Touchless user experience.

## Technologies Used
- Python
- OpenCV
- MediaPipe

## Future Improvements
- Add more gesture controls.
- Improve gesture recognition accuracy.
- Extend the project with additional touchless features.
